## About

I'm a final-year **iMSc.IT** student (Semester 8) with **6 months of hands-on experience** as an AI/ML Intern. I work at the intersection of machine learning, computer vision, and generative AI — building systems that turn raw data into real-world impact.

My core interests lie in **deep learning architectures**, **healthcare AI**, **NLP/LLMs**, and scalable ML pipelines. I enjoy working end-to-end — from data wrangling and model training to deployment and monitoring.

> Currently exploring: **LLM fine-tuning · RAG pipelines · MLOps · Multimodal AI**
